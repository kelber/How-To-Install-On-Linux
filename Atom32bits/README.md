# How to install atom.io 32bits

```
sudo add-apt-repository ppa:webupd8team/atom
sudo apt install atom
sudo apt-get update
```


# Updating your angular-cli

```
npm uninstall -g angular-cli
npm cache clean
npm install -g angular-cli@latest
```
